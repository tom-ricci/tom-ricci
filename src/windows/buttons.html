Notes

Instead of using a traditional JS framework, I built this site with HTMX and HyperScript, using vanilla JS where necessary. I ran into a few tasks which would've been better suited for a traditional framework—maybe I'll create my own HTMX-compatible library to solve these? Regardless, it was a pleasure to step back from complexity and rely on hypermedia controls. Hopefully I can take this approach again for another project!
